# R02 人工链接与页面检查清单

## 启动方式

- [ ] 直接打开项目根目录中的 `index.html`。
- [ ] 核心页面不依赖网络、npm 或构建工具。

## 三级路径

- [ ] 首页 → Archive。
- [ ] Archive → TYPE → A01。
- [ ] `index.html → archive.html → collections/type.html → entries/a01-nonhuman-alphabet.html` 可以连续点击。

## Archive

- [ ] Archive → TYPE。
- [ ] Archive → IMAGE。
- [ ] Archive → SYSTEM。
- [ ] Archive → FIELD NOTES。
- [ ] Archive → A01—A12 每个详情页。
- [ ] ALL / TYPE / IMAGE / SYSTEM / FIELD NOTES 筛选正常。
- [ ] GRID / LIST 正常。
- [ ] INDEX / MIX 正常。

- [ ] TYPE → A01、A02、A03。

## Index

- [ ] Index → A01—A12 每个详情页。
- [ ] Index 可以返回首页和 Archive。

## 详情页

- [ ] 每个详情页 → 所属分类。
- [ ] 每个详情页 → Archive。
- [ ] A01 上一篇 A12、下一篇 A02。

- [ ] 每页四张详情图属于正确编号。

- [ ] About 内容、Archive 和 Contact 入口正常。

- [ ] Contact 可以返回首页、About 和 Archive。
- [ ] 404 页面可以返回首页和 Archive。

## 资源与控制台

- [ ] CSS 没有丢失。
- [ ] 两个本地字体路径有效。
- [ ] 封面和详情图片没有丢失。
- [ ] 图片没有被拉伸。
- [ ] 浏览器控制台没有未处理错误。

## 响应式

- [ ] 375px：单列内容、导航可用、无横向滚动。
- [ ] 768px：分类卡片和详情布局正常。
- [ ] 1024px：导航、网格和详情首屏正常。
- [ ] 1440px：内容宽度和留白正常。
- [ ] 键盘焦点清楚。
- [ ] 减少动态偏好生效。
