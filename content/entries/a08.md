## A08 SIGNAL SPECIMENS｜信号标本

**项目类别：SYSTEM｜视觉系统实验**

### 1. 中文项目简介

本项目将波形、噪声、条码、频谱与干扰信号视为可采集、切片、编号和保存的视觉标本。通过截取不同信号片段，建立采样坐标、强度等级、频率区间与保存状态等记录规则，并将其整理为连续图版、索引卡和分类页面。视觉结果介于技术档案与自然史图谱之间，试图讨论不可见的信息如何在被测量、命名和归档之后，获得类似实体材料的秩序与可信度。

### 2. English Project Description

This project treats waveforms, noise, barcodes, spectra, and interference as visual specimens that can be collected, sectioned, numbered, and preserved. Signal fragments are sampled and organized through a set of archival rules, including coordinates, intensity levels, frequency ranges, and preservation states. The resulting system appears across specimen plates, index cards, and classified pages, positioned between a technical record and a natural-history atlas. It examines how invisible information acquires the order and apparent materiality of a physical object once it is measured, named, and archived.

### 7. 四张内容图图注

**整体系统：**
不同来源的信号片段被纳入统一网格，以编号、频段和状态构成分类总览。

**材料细节：**
放大后的噪点、断裂条码与频谱边缘，显示信号在转译过程中留下的物质痕迹。

**过程推演：**
从原始采样、波形切片到编码归档，逐步测试分类规则与版面层级的稳定性。

**应用或现场：**
标本图版、索引卡与屏幕界面共同陈列，使静态档案与实时信号形成对照。

### 8. 笔记

先把信号当作图像观察，切片之后，相似的波形会因为尺度、密度和断裂方式呈现出明显差异。在编号和分组时不断调整规则，尽量让系统看似严密，却仍保留噪声本身的不稳定性。
