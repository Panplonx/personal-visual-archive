## A07 FALSE TAXONOMY｜虚构分类学

**CATEGORY｜项目类别**
SYSTEM｜视觉系统实验

### 1. 中文项目简介

以不存在的标本为研究对象，依据轮廓、表面纹理、附着结构与损坏程度建立编号、层级和分支规则。项目通过图谱、标签、关系图与保存记录，将形态信息整理为一套自成体系的视觉系统。最终结果介于自然史档案与制度化图表之间，用看似可信的秩序讨论分类如何制造知识、权威与排除。

### 2. English Project Description

A07 FALSE TAXONOMY constructs a visual system for specimens that do not exist. Each object is assigned an index, morphological description, branch position, and preservation status according to rules that appear consistent but remain deliberately absurd. Charts, labels, diagrams, and archival records organize contradictory traits into a credible pseudo-scientific order. Positioned between a natural history catalogue and an administrative classification system, the project examines how naming, grouping, and visual authority can make invented knowledge appear stable, objective, and legitimate.

### 3. 一句话摘要

为不存在的标本建立可信分类系统，观察制造知识权威的流水线工程。

**整体系统**
整体系统呈现标本编号、层级分支与保存状态之间的对应关系，形成可检索的伪档案结构。

材料细节集中记录表面纹理、边缘损伤、附着物与标签痕迹，作为分类判断的视觉依据。

**过程推演**
过程推演展示形态特征如何被拆分、比较并重新归组，同时保留规则细分与修订记录。

**应用或现场**
应用或现场将分类图谱、标本卡与索引标签并置，模拟一套可被查阅和维护的档案环境。

### 8. 笔记

先为每件事实上不存在的标本设定编号，再从轮廓、纹理、损坏和保存方式中提取特征。实际编排时，发现规则越细，例外就越多，因此保留了部分矛盾和修订痕迹。
